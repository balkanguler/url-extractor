# url-extractor
Extracts URL from image and redirects to this URL


Uses these environment variables to call AWS services.

AWS_ACCESS_KEY_ID
AWS_SECRET_ACCESS_KEY
AWS_REGION

in your local aws_config.json file can be used. Its structure:
{
    "accessKeyId": "...",
    "secretAccessKey":  "...",
    "region": "us-east-1"
}
